# 桜町中学校1976年卒業同窓会サイト テスト手順書

## 🚀 サイト起動方法

### 方法1: Pythonサーバー（推奨）
```bash
# プロジェクトディレクトリに移動
cd /home/tshirai/learning/hp

# ローカルサーバーを起動
python3 -m http.server 8000

# ブラウザで以下にアクセス
http://localhost:8000
```

### 方法2: Live Serverや他のサーバー
任意のHTTPサーバーでディレクトリをホストしてください。

## ⚠️ 既知の制限事項

### 簡易版 (`members-simple.html`)
- データは一時的（ページリロードで消失）
- 写真アップロードはテスト機能のみ
- Firebase機能は無効

### 拡張版 (`members-enhanced.html`)
- Firebase Console設定が必要
- インターネット接続が必要

## 🔧 トラブルシューティング

### CSSが読み込まれない
- パスが正しいか確認: `assets/css/styles.css`
- ファイルが存在するか確認

### JavaScriptエラー
- ブラウザのコンソールでエラーを確認
- ローカルサーバーを再起動

### ページが表示されない
- サーバーが起動しているか確認
- ポート8000が使用可能か確認
